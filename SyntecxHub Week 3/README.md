# E-commerce Conversion Funnel Analysis

## Project Overview

This project is a complete Sales Funnel Analysis performed using Python to evaluate customer movement through different stages of a sales funnel. The objective is to identify where customers drop off, measure conversion performance, and generate actionable business insights that can help improve overall sales efficiency.

The project follows the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), KPI calculations, visualization, and business recommendations.

This project was completed as **Week 3 Task** during my **Data Analytics Internship at SyntecxHub**.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Jupyter Notebook
- Power BI

## Problem Solved

Businesses often invest heavily in attracting website visitors but fail to understand why only a small percentage eventually become paying customers.

This project answers important business questions such as:

- Which stage of the sales funnel loses the most customers?
- What is the conversion rate between each stage?
- How efficient is the entire funnel?
- Which customer acquisition channels perform best?
- What strategic improvements can increase conversions?
